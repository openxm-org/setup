Asir for iSH Shell/iOS

This is a binary of Risa/Asir (version 20220911) for iOS.
This binary was checked on iPadOS15.7(iPad Pro 12.9-inch gen5).

  • binary asir_ios.tar.gz including fep(a frontend for asir) and PARI/GP(a computer algebra system for number theory).
  • screenshot on iPad Pro 12.9-inch gen5

Installation:

  1. Install iSH Shell to your iPhone or iPad through AppStore.
  2. Open iSH and update packages as follows:
    # apk update
    # apk upgrade
  3. Execute the following on iSH:
    # wget https://openxm-org.github.io/setup/ios/install.sh
    # sh install.sh
    # exit
  4. Re-open iSH and execute the following:
    # asir
